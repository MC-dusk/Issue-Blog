# 对抗微软GIF输入指南

对win11输入法emoji输入面板强制提供的gif服务不爽，改host文件，解决。

由于gif服务是由闪萌提供的，屏蔽其域名即可。

```
0.0.0.0 weshineapp.com
0.0.0.0 www.weshineapp.com
0.0.0.0 api.open.weshineapp.com
0.0.0.0 dl.weshineapp.com
0.0.0.0 dl1.weshineapp.com
0.0.0.0 dl2.weshineapp.com
0.0.0.0 dl3.weshineapp.com
0.0.0.0 dl4.weshineapp.com
0.0.0.0 dl5.weshineapp.com
0.0.0.0 dl6.weshineapp.com
```

如果使用代理，host文件将失效，需要在代理规则屏蔽对应域名。

```
DOMAIN-SUFFIX,weshineapp.com,REJECT
```
