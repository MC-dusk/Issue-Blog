# 清除Windows Defender保护历史记录

Windows 安全中心，这么久了竟然连清理历史的功能都不做，太离谱了。 这个链接 (opens new window)里面提供了一个工具 (opens new window)，也许有帮助。 原来早就有这个问题 (opens new window)了。

原理倒不复杂，删除下面两个路径的文件就行了。

快速扫描记录
C:\ProgramData\Microsoft\Windows Defender\Scans\History\Results

保护历史记录
C:\ProgramData\Microsoft\Windows Defender\Scans\History\Service\DetectionHistory

但是会遇到权限问题，提权管理员权限也没用，拒绝访问。

下面是文心一言的回答，仅供参考。

要删除 Windows Defender 的保护历史记录，可按照以下分步操作进行，推荐优先使用手动清理文件夹的方法：

# 方法一：手动删除历史记录文件夹（推荐）

  1. 显示隐藏文件和文件夹

    • 打开 文件资源管理器（快捷键 Win + E），进入 C盘。
    • 点击顶部菜单栏的 “查看”，勾选 “隐藏的项目”（或通过 “选项” → “查看” 勾选 “显示隐藏的文件、文件夹和驱动器”）。

  2. 定位到历史记录文件夹

    • 导航至路径：
      C:\ProgramData\Microsoft\Windows Defender\Scans\History
    • 若无法直接打开 Scans 文件夹，可尝试 进入安全模式（重启时按住 Shift 键，选择 “疑难解答” → “高级选项” → “启动设置” → 重启后按 F4 进入安全模式），再操作删除。

  3. 删除关键文件夹

    • 删除以下文件夹（若存在）：
      • Quick（快速扫描记录）
      • Resource（资源扫描记录）
      • Service\DetectionHistory（威胁检测历史）
    • 注意：删除前建议备份重要数据，避免误操作。

  4. 重启系统

    • 重启电脑后，Windows Defender 的保护历史记录将被清空。

# 方法二：使用 PowerShell 命令（自动清理）

若希望设置自动清理规则（如每 7 天清除一次），可按以下步骤操作：

  1. 以管理员身份打开 PowerShell（右键开始菜单选择 “Windows PowerShell（管理员）”）。
  2. 输入以下命令并回车（将 N 替换为天数，例如 7）：
    Set-MpPreference -ScanPurgeItemsAfterDelay N
    • 此命令会设置扫描记录在 N 天后自动删除。

# 方法三：通过事件查看器清理（补充）

  1. 按 Win + S 搜索 “事件查看器” 并打开。
  2. 导航至：
    应用程序和服务日志 → Microsoft → Windows → Windows Defender → Operational
  3. 右键点击 “Operational”，选择 “清除日志”，确认删除即可。

# 注意事项

  1. 权限问题：若文件夹无法删除，可能是系统权限限制。可尝试：
    • 以管理员身份运行文件资源管理器。
    • 使用 PE 系统（如微PE）启动后删除文件（适用于极端情况）。
  2. 第三方工具风险：避免使用非官方工具清理，可能携带恶意软件。
  3. 系统影响：删除记录不会影响防护功能，但会丢失历史威胁信息，建议定期清理以释放空间。

# 总结

  • 推荐方法：手动删除 History 文件夹下的内容（方法一），操作简单且彻底。
  • 进阶需求：使用 PowerShell 设置自动清理（方法二），适合长期维护。
  • 补充清理：通过事件查看器删除日志（方法三），可进一步释放空间。

按需选择方法，完成后重启系统即可生效。
