# Github镜像

# 下载加速

基本都是这个仓库 (opens new window)的镜像实例，使用方式均为在原链接前面加上加速链接，比如https://ghp.ci/https://github.com/hunshcn/gh-proxy/blob/master/README.md，下载对应文件，即使是展示页

  1. https://ghproxy.com/ (opens new window)（喜提gfw）
    • https://mirror.ghproxy.com/ (opens new window)
    • https://ghp.ci/ (opens new window)（新域名）
  2. https://hub.gitmirror.com/ (opens new window)（跳转到这 (opens new window)）
    • raw文件可替换域名为raw.gitmirror.com来加速下载
  3. https://ghproxy.cn/ (opens new window)
    • https://ghproxy.cc/ (opens new window)（与ghproxy.cn同源）
  4. https://ghproxy.net/ (opens new window)
  5. https://gh-proxy.com/ (opens new window) slow
  6. https://ghps.cc/ (opens new window) blocked
  7. https://gh.ddlc.top/ (opens new window) cf limit
  8. https://d.serctl.com/ (opens new window)（只能进站下载，不能拼写直链）

# 镜像站

mirror镜像可将github.com替换为对应的顶级域来访问镜像站，比如github.site

  1. 在任意 github 链接后面加上 fast，也就是https://githubfast.com (opens new window)，示例 (opens new window)（仅限git）
  2. 在任意 github 链接前面加上 521，也就是https://521github.com (opens new window)，示例 (opens new window)（已失效）
  3. https://kkgithub.com/ (opens new window)
  4. https://github.site/ (opens new window)（与ghproxy.cn同源）
  5. https://github.store/ (opens new window)（与ghproxy.cn同源）
